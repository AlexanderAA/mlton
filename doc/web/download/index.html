The latest public release of MLton, 20030716, is available in the following formats.

  • Binary
    • rpm for Red Hat 7.x, 8.x, 9.x
    • deb for Debian GNU/Linux sarge (libc6 >= 2.3.1-1, libgmp3-dev >= 4.0.1)
    • tgz for Linux (glibc 2.2)
    • tgz for Cygwin/Windows
    • tgz for FreeBSD
    • tgz for Sparc/SunOS 5.7, 5.8
  • Source
    • tgz for any platform
    • srpm for Red Hat 7.x, 8.x, 9.x (rpm 4.x)
You can also access the MLton sources in the following ways.
  • on the web with ViewCVS
  • via anonymous CVS with
    cvs -d :pserver:anonymous@cvs.mlton.org:/cvsroot/mlton co mlton
You can also download newer, experimental versions of MLton.
You can also download other MLton packages for Debian GNU/Linux.
