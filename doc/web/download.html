The latest version (VERSION) is available in the following formats.
  • Binary RPM (4.4M)
    • rpm for Red Hat 7.x (glibc 2.2, rpm 4.x)
    • rpm for Red Hat 6.x (glibc 2.1, rpm 3.x)
  • Source RPM (3.7M)
    • rpm for Red Hat 7.x (rpm 4.x)
    • rpm for Red Hat 6.x (rpm 3.x)
  • Gzipped tar (for non-RPM distributions)
    • binary tgz for Cygwin/Windows
    • binary tgz for Red Hat 7.x (glibc 2.2)
    • binary tgz for Red Hat 6.x (glibc 2.1)
    • source tgz
You can also view older and newer, experimental versions of MLton.

You can access our current internal version of MLton via anonymous CVS as follows:
cvs -d :pserver:anonymous@cvs.mlton.org:/cvsroot/mlton co mlton

You can also browse the MLton sources using ViewCVS.
