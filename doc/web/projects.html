Here is a list of projects that use or have used MLton. If you would like to add something to the list, please send mail to MLton@sourcelight.com with a brief description and a link.
  • Dan Wang has implemented a prototype for Type-Preserving Garbage Collectors, an approach to generating a strongly typed garbage collector as part of a program.
  • Ánoq of the Sun has used MLton as a crosscompiler from Linux to Windows for the development of commercial software at his company, Hardcore Processing, including
    • CEX3D Converter, a conversion program for 3D objects.
    • various graphics games
  • Tom 7 is writing replacements for standard network services (fingerd, ftpd, identd, ...) using SML and MLton.
  • Automatic Code Generation from Coloured Petri Nets for an Access Control System
