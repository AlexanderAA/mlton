Here is a list of projects that use or have used MLton. If you would like to add something to the list, please send mail to MLton at mlton.org brief description and a link.
  • Ánoq of the Sun uses MLton as a crosscompiler from Linux to Windows for the development of commercial software at his company, Hardcore Processing, including
    • CEX3D Converter, a conversion program for 3D objects.
    • Interactive Showreel, which contains a crossplatform GUI-toolkit and a realtime renderer for a subset of RenderMan written in Standard ML.
    • various graphics games
  • Dan Wang uses MLton to implement a prototype Type-Preserving Garbage Collector, which generates a strongly typed garbage collector as part of a program.
  • Kjeld Mortensen uses MLton for Automatic Code Generation from Coloured Petri Nets for an Access Control System.
  • PolySpace Technologies uses MLton to compile their commercial product for detection of runtime errors in embedded systems based on abstract interpretation.
  • Simon Helsen uses MLton to develop MLOPE, an offline partial evaluator for full Standard ML.
  • Tom 7 uses MLton to implement replacements for standard network services (fingerd, ftpd, identd, ...).
